# Singlish Chat Translator - Automated UI Testing

This repository contains an automated UI testing suite built with Python and Playwright. It evaluates the transliteration accuracy of a Sinhala-English (Singlish) chat translator web application using extreme and edge-case inputs.

The automation script reads test cases from an Excel file, interacts with the frontend UI, retrieves the generated Sinhala output, and writes the results back to the Excel file.

## Prerequisites

Before setting up the project, ensure you have the following installed:
* **Python 3.11 or 3.12** (Ensure Python is added to your system PATH during installation).
* **Google Chrome** (Recommended) or Chromium.
